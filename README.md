## Инструкция по сборке и запуску

### Требования:
- Java 17 или выше
- Maven 3.6 или выше

### Сборка проекта:

1. **Клонируйте или создайте проект** с указанной структурой файлов

2. **Выполните сборку**:
```bash
mvn clean package
```

3. **Запустите приложение**:
```bash
java -jar target/nth-min-service-1.0.0.jar
```

### Альтернативный запуск (без сборки JAR):
```bash
mvn spring-boot:run
```

### Использование:

1. **После запуска** откройте браузер и перейдите по адресу:
```
http://localhost:8080/swagger-ui.html
```

2. **В Swagger UI** используйте endpoint:
```
GET /api/numbers/exc-min
```

3. **Параметры**:
   - `filePath`: абсолютный путь к Excel файлу (например, `C:/data/numbers.xlsx`)
   - `n`: позиция минимального числа (например, 3 для третьего минимального)

### Пример тестового файла:

Создайте Excel файл `numbers.xlsx` с числами в первом столбце:
```
10
5
8
3
15
2
20
```

### Пример запроса:
```
GET http://localhost:8080/api/numbers/exc-min?filePath=C:/data/numbers.xlsx&n=3
```

**Ответ**:
```json
{
  "MinNumber": 5,
  "status": "success",
  "message": "Найдено 3-ное минимальное число: 5"
}
```
